Meeting - Attendee Meeting - Attendee

.

Description

This view allows you to create a new Meetings Attendee.

If you create a user with the role 'Speaker' or 'Chair', you can used them as speakers in Agendaitem.

Controls

'(User) Id' -- The id of the Meeting Attendee. This is the last part of the URL that is the address of the attendee.

'Surname' -- The surname of the attendee

'First names' -- The first names of the attendee.

'Organisation' -- The name of the organization of the attendee.

'Email' -- The email address of the attendee.

'Telephone' -- The telephone number of the attendee.

'Postal Address' -- The postal address of the attendee.

'Attending' -- One of the state of the attendee('yes','tentative','no, but keep me informed').

'Verified' -- ??

'Role' -- The role of the attendee. The role is given by Manager only.

'Resume' -- short resume of the attendee. Given by Manager only.
