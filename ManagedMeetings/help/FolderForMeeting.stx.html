Meeting - Folder for Meetings Meeting - Folder for Meetings

.

Description

This view allows you to create/edit a new Folder for Meetings.

Inside the newly created meeting, an index_html and an past_html objects are added. By default the index_html display the future events and past_html the past/cancelled/tentative events.
You can choose to modify the content.

Controls

'Id' -- The id of the Folder for Meetings. This is the last part of the URL that is the address of the folder for meetings.

'Meeting Folder Title' -- (optional) The title of the folder.

'Summary' -- (optional) A short description of the folder

'Possible Event types' -- (optional) Contains the possible event types of events that you intend to add to this folder..If you leave empty then the folder will allow any type.

'Mail enconding' -- The encoding used for the website's pages (usually set on the standard header as 

RESPONSE.setHeader('content-type', 'text/html;charset=utf-8')
). This is used to encode the email body. If empty iso-8859-1 is consider.

*Methods*

open_rss() -- generate a RDF file from all the meetings contains by this folder.
